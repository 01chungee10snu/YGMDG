- `scripts/qa_playwright.py`
  - MacBook의 `/Applications/Google Chrome.app`을 직접 사용해 실제 화면/콘솔/퀴즈 플로우 검증
- `.openclaw/harness/orchestrator.md`

## 남은 리스크

- 공개 배포 대상 설정(`package.json`, `vercel.json`, `netlify.toml` 등)이 없어 원격 배포 검증은 수행하지 않았음.
- Google Sheets 실제 POST는 endpoint/권한 의존성이 있어 이번 로컬 QA에서는 fallback 가능 구조만 범위에 포함했음.
