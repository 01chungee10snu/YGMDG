# 용강 만들기 프로젝트 운영 지침

이 레포에서 대규모 게임 업데이트, 이미지 자산 생성, Google Sheets DB 연동, QA를 수행할 때는 `.openclaw/harness/orchestrator.md`를 먼저 따른다.

## 현재 하네스
- 위치: `.openclaw/harness/`
- 목적: 제철 밸류체인 기반 머지 게임을 플레이 가능한 정적 웹 게임으로 유지한다.
- 검증: 정적 문법 검사, 로컬 서버 브라우저 smoke test, Matter.js 런타임 오류 확인.
